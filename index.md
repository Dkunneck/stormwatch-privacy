# StormWatch Privacy Policy

StormWatch is a personal weather-analysis system used for storm chase planning.

## What data is used

When the GPT Action is used:
- Requests may include:
  - action names (e.g., spc_fetch, spc_run)
  - output directory paths
  - optional workflow parameters

## How data is used

- Requests are sent to a locally hosted StormWatch API.
- The API executes deterministic workflows and writes results to local files.
- Data is not used for advertising, tracking, or resale.

## Storage

- Data is stored only on the operator’s local StormWatch system.
- No external database or third-party storage is used.

## Sharing

- No user data is shared with third parties.
- The API is protected by a bearer token.

## Security

- The API requires authentication.
- The service is intended for controlled, temporary use.

## Contact

This is a personal project. For questions, contact the project owner.
